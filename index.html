🧬 CRISPR-Phage Resistance Network Analyzer

Detect CRISPR arrays in bacteria, extract spacers, align to phage genomes, and visualize resistance networks

📖 User Guide ▼

This tool runs entirely in your browser — no data is uploaded to any server. All computation happens locally.

What does this tool do?

This tool takes bacterial and phage genome sequences (FASTA format) and:

  1. Scans bacterial genomes for CRISPR arrays (direct repeats flanking spacers)
  2. Classifies the Cas system type (Type I–V) from each bacterial genome
  3. Extracts all spacer sequences from detected CRISPR arrays
  4. Aligns spacers against phage genomes using Smith-Waterman local alignment
  5. Checks for a valid PAM sequence flanking each alignment hit
  6. Computes a resistance score (alignment + PAM) and draws an interactive network

Input files

  • Format: .fasta, .fa, or .fna
  • One file = one organism. Multi-contig assemblies are supported — all contigs in one file are treated as one organism.
  • Upload one or more bacterial files and one or more phage files.

Step-by-step

  1. Upload your bacterial FASTA files (left panel)
  2. Upload your phage FASTA files (right panel)
  3. Adjust parameters if needed (see below)
  4. Click Analyze
  5. Explore the network — hover over nodes for details, drag to rearrange
  6. Download the network as SVG or the full results as CSV

Parameters

ParameterDescriptionDefault

Reading the network

  • ● Green nodes = bacteria
  • ● Red nodes = phages
  • Distance encodes resistance: nodes that are farther apart have a higher resistance score (the bacterium is more likely to resist that phage)
  • Hover over any node to see details (CRISPR arrays found, Cas type, spacer count)
  • Drag nodes to rearrange the layout

PAM inference

The PAM motif is automatically selected based on the detected Cas type:

Cas typePAM motifPosition
Type II (Cas9)NGGDownstream of protospacer
Type V (Cas12)TTNUpstream of protospacer
Type I (Cascade)AWGUpstream of protospacer
Type III (Cas10)RAAGDownstream of protospacer

Outputs

  • Network SVG: vector image of the resistance network, suitable for publications
  • Results CSV: full table with every spacer-phage alignment, PAM status, and scores
Tip: For best results, use complete or near-complete genome assemblies. Very fragmented assemblies (>500 contigs) may miss CRISPR arrays that span contig boundaries.
